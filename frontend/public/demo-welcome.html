🎙️ Talk to Me — Voice

Click the microphone icon next to the text input to use voice. This demo uses the browser's built-in Web Speech API for voice-to-text — it's functional but basic.

In a full deployment, Autonomy integrates with enterprise voice services for higher accuracy, multi-language support, and natural conversation flow.

To enable voice in this demo:

Browser:
Chrome, Edge, or Safari (Firefox has limited support)
Permission:
Allow microphone access when prompted by the browser
Settings:
Chrome → Settings → Privacy → Site Settings → Microphone → Allow for this site
HTTPS:
Required — voice only works over HTTPS (this demo site uses it automatically)
